About Pinkachu

Pinkachu website project helped our group learned the basic elements of HTML, such as headings, paragraphs, links, and images. We used these elements to organize the content properly and added CSS to design and style our website. Through this activity, we understood how HTML creates the structure of a web page and how CSS helps show our ideas and aesthetics in the browser. This project represents our whole group..
